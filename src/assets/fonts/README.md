# Font Sources

These local webfont files are bundled so the record card renders consistently offline and in exported images.

- NanumSquare Neo: NAVER Nanum font family, downloaded from `https://cdn.jsdelivr.net/gh/moonspam/NanumSquareNeo@1.0/`
- Pretendard: SIL Open Font License 1.1, downloaded from `https://github.com/orioncactus/pretendard`
- Paperlogy/Paperozi: SIL Open Font License 1.1, downloaded from Noonnu's jsDelivr source at `https://noonnu.cc/font_page/1456`
- Gmarket Sans: SIL Open Font License 1.1, downloaded from Gmarket/Noonnu distribution at `https://corp.gmarket.com/fonts/`
- SUIT: SIL Open Font License 1.1, downloaded from `https://github.com/sun-typeface/SUIT`
- LINE Seed Sans KR: SIL Open Font License 1.1, downloaded from the `@kfonts/line-seed-sans-kr` webfont package
- Noto Sans KR: SIL Open Font License 1.1, downloaded from Google Fonts. This project bundles the needed local TTF weights because the Google Fonts web CSS shards Korean WOFF2 across many unicode-range files.

Do not sell the font files themselves. Keep the original license terms with redistributed builds when required.
